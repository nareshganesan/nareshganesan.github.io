1000Lookz - Virtual Tryon

Sun, Aug 14, 2016 ( CV )

Its been couple of months since I started working for 1000Lookz/VDime. The journey has been really wonderful so far. Everyday I get to work on cool technologies. Python has been the bread and butter @ 1000Lookz. We are company that specialises in CV, machine learning and Deep Learning. Some of the day to day techology we use @ 1000Lookz have been listed below.

Technologies

DL - Caffe

CV - Dlib - OpenCV - Tesseract - Ocropy

Tools
